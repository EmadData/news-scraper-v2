# Multi-News Scraper with Python

This project is a web scraping script that collects news from multiple Iranian news websites.

## Features
- Scrapes news titles and full text from different sites
- Categorizes news by topic
- Removes duplicates and short texts
- Handles missing data safely
- Saves data to CSV file
- Uses **JSON config** to store site URLs and categories, making it easy to add new sources

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas
- JSON
